3. Penggunaan Data

  • Tidak ada data disimpan di server.
  • Tidak ada pelacakan (tracking) menggunakan cookie, Google Analytics, atau pihak ketiga.
  • Snapshot dan rekaman video/audio hanya disimpan sementara di memori browser Anda dan langsung diunduh ke perangkat Anda saat Anda memilih untuk menyimpannya.
  • File yang diunduh tidak pernah dikirim ke server eksternal.

4. Keamanan

  • Aplikasi ini berjalan 100% di sisi klien (client-side only).
  • Tidak ada koneksi ke server eksternal selain untuk memuat aset statis (CSS, JS, ikon).
  • Semua aliran media (video/audio) diproses secara lokal menggunakan WebRTC dan Web Audio API.

5. Rekaman dan Snapshot

Saat Anda merekam video atau mengambil snapshot, file tersebut:

  • Dibuat di browser Anda menggunakan MediaRecorder API.
  • Disimpan sementara di memori sebagai Blob.
  • Diunduh langsung ke perangkat Anda melalui tautan unduhan.

File tersebut tidak pernah meninggalkan perangkat Anda.

6. Peramban dan Izin

AV Test Studio Pro hanya berfungsi di browser modern yang mendukung:

  • getUserMedia() API
  • MediaRecorder API
  • Web Audio API

Anda dapat mengelola izin kamera/mikrofon kapan saja melalui pengaturan browser Anda.
